# Interactive Singly Linked List (C Implementation)

## 📌 Overview
This repository features a menu-driven C application for demonstrating dynamic memory management and core data structure operations on a **Singly Linked List**. It utilizes nested `switch-case` structures to allow interactive user control via a Command Line Interface (CLI).

---

## ✨ Features

- **Insertion:**
  - Beginning (Head)
  - End (Tail)
  - Arbitrary Position (Index-based)
- **Deletion:**
  - First Node
  - Last Node
  - Specific Index
- **Traversal & Search:**
  - Full list rendering with data and memory pointer addresses
  - Linear search function returning node indices
  - Dynamic node counter

---

## 🚀 How to Run

### Prerequisites
A C compiler (e.g., `gcc` or `clang`).

### Compilation & Execution (via Terminal or Git Bash)

```bash
# Compile the program
gcc linked_list.c -o linked_list

# Run the program
